    # VideoPlayer 视频播放器

    # 引入方式

    import { VideoPlayer } from 'yike-app-ui';

Vue.component(VideoPlayer.name, VideoPlayer);

    TIP

    由于本组件使用了rem,但这个项目没有设置根元素的字体。导致样式无法观看，先凑合看看吧。 目前本组件使用的是系统全屏，如果在手机端使用请把useNativeFullScreen设置为false;

    WARNING

    ios自动播放功能，要加入版本控制。一课 2.12.8 ykvc 57 作业帮 11.6.6 zybvc 71

    # Attributes

    参数 说明 类型 默认值 版本
    src 视频地址 String - -
    poster 视频封面 String - -
    autoplay 视频自动播放 Boolean false -
    preload 视频预加载(wifi下开启预加载 4G下不开启。因为vivo、oppo两款手机预加载无效，播放时还是会重新请求视频资源，所以设置为meta) String auto -
    videoTitle 视频标题 String - -
    playIconPosition 封面播放按钮的位置 Object {top:'50%',translateY: '-50%'} -
    objectFix 视频填充方式，同object-fit的属性值 String contain -
    useNativeFullScreen 调用系统全屏 Boolean false -

    # methods

    方法名 参数 返回值 介绍
    videoPlay 无 无 播放视频
    videoPause 无 无 暂停播放视频

    # callback

    回调函数 说明 参数
    onPlaySuccess 播放成功后执行 默认无
    onBeforeEnterFullScreen 进入全屏前执行 -
    onBeforeExitFullScreen 退出全屏前执行 -
    onHideControls 控制栏隐藏 -
    onShowControls 控制栏显示 -
    onTouchStartTimeLine 拖动进度条 -

    # Event

    事件名 说明 参数 值
    getVideoStatus 当前视频的播放状态 无 playing or ended
    controlsStatus 控制栏的状态 - true or false

    # Slot

    name 说明 版本
    playButton 替换封面上播放按钮 0.0.16
